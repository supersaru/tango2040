# Halogen32

Basic arm MCU development board design with USB-C, STEMMA-QT, and SWD debugging. Designed to mimic Adafruit feather form factor.

- Features
  - [x] STM32F411
  - [x] Boot switch
  - [x] Reset Switch
  - [x] USB-C
  - [x] SWD programming
  - [x] Neopixel status LED
  - [x] STEMMA-QT
  - [ ] GPIO
  - [ ] Battery Managment
  - [ ] PCB Design
